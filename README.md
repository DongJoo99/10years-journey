### 🛠 오늘 한 일 (상세)

#### 1. Git 설치 및 환경 설정
- Windows 환경에서 Git 설치 진행
- 설치 과정 중 다음 설정을 선택함:
  - Git from the command line and also from 3rd-party software
  - Use Visual Studio Code as Git's default editor
  - Use bundled OpenSSH
  - Use the OpenSSL library
  - Checkout Windows-style, commit Unix-style line endings
- Git Bash 정상 실행 확인

#### 2. GitHub 저장소 생성
- 저장소 이름: `10years-journey`
- Public 저장소로 생성
- README 파일 포함하여 초기화

#### 3. 로컬 PC와 GitHub 연결
- Git Bash에서 로컬 디렉터리 생성
- `git init`으로 로컬 저장소 초기화
- GitHub 원격 저장소(origin) 연결
- `git pull origin main`으로 README 파일 가져오기

#### 4. 개발 도구 준비

### 📚 오늘 배운 것

- Git은 단순한 백업 도구가 아니라 **변경 이력을 관리하는 시스템**
- GitHub는 코드 저장소이면서 동시에 **포트폴리오 공간**
- 처음 환경 설정이 귀찮지만, 이 단계를 넘기면 이후가 훨씬 편해진다
- 바탕화면 경로 문제처럼 사소한 오류도 차분히 보면 해결 가능하다
