# 2026-03-26 (Day 25)

## 오늘 한 것

오늘은 문제를 맞추는 것보다 면접에서 바로 말할 수 있도록 답변 구조를 만드는 연습을 했다.

문제 유형, 주원인, 판단 근거, 한 줄 요약으로 정리하는 방식으로  
어떤 상황에서도 일정한 구조로 설명할 수 있도록 훈련했다.

CPU Ready, Disk latency(KAVG), Memory Swap 문제를 각각 빠르게 판단하고  
구조에 맞게 설명하는 연습을 반복했다.

특히 고민하지 않고 바로 답변이 나올 수 있도록  
속도와 정확도를 함께 높이는 데 집중했다.

---

## 기억할 것

- 문제 유형 → 주원인 → 근거 → 한 줄 요약 구조 유지
- CPU idle 높고 Load 높으면 CPU Ready
- await 높고 util 낮으면 KAVG
- available 낮고 si/so 발생하면 Swap
- Memory 문제는 Disk 문제처럼 보일 수 있음

---
