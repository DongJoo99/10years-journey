# 2026-03-09 (Day 9)

## 오늘 한 것

오늘은 Linux 메모리 구조와 Swap에 대해 공부했다.

free -h 명령어를 통해 현재 메모리 상태를 확인하는 방법을 실습했다.

특히 free 값보다 available 값이 실제 사용 가능한 메모리라는 것을 알게 되었다.

또 Swap이 단순히 메모리가 부족할 때만 사용하는 것이 아니라
Linux 커널 정책에 따라 오래 사용하지 않는 메모리를 Swap으로 이동시킬 수도 있다는 것도 이해했다.

swappiness 값이 Swap 사용 정책을 결정한다는 것도 처음 알게 되었다.


## 기억할 것

- free보다 available을 확인한다
- Swap 사용 = 항상 메모리 부족은 아니다
- Swap 사용이 많아지면 서버 성능이 급격히 저하될 수 있다
- swappiness 값으로 Swap 정책을 조절할 수 있다


## 느낀 점

CPU와 Load Average를 공부한 뒤 메모리까지 같이 보니까
서버 상태를 어떻게 판단해야 하는지 조금씩 연결되는 느낌이 들었다.

특히 Swap이 많아지면 서버가 매우 느려질 수 있다는 점이 인상적이었다.
